# โปรแกรมจัดการงานในมือแบบออฟไลน์

เปิดใช้งานบนเครื่อง Windows ได้โดยดับเบิลคลิกไฟล์ `start-task-manager.bat`

อีกวิธีคือเปิดไฟล์ `task-manager.html` ด้วยเบราว์เซอร์โดยตรง เช่น Chrome, Edge หรือ Firefox

ข้อมูลจะถูกบันทึกไว้ในเครื่องนี้ผ่าน `localStorage` ของเบราว์เซอร์ ไม่ต้องใช้ฐานข้อมูล ไม่ต้องใช้อินเทอร์เน็ต และไม่ต้องเปิดเซิร์ฟเวอร์

## สำรองข้อมูล

- กด `ส่งออก` เพื่อบันทึกงานเป็นไฟล์ `.json`
- กด `นำเข้า` เพื่อโหลดไฟล์สำรองกลับเข้ามา

ควรส่งออกไฟล์สำรองเป็นระยะ โดยเฉพาะก่อนล้างข้อมูลเบราว์เซอร์หรือย้ายเครื่อง
